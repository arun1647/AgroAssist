## 📌 Overview

Agro Assist Farm is a modern agriculture-focused web application built to help farmers and agricultural users make better decisions using technology. The platform provides crop recommendations, seasonal farming insights, market information, image-based analysis, and multilingual support through a responsive user interface.

---

## 🚀 Key Features

- 🌾 Intelligent crop recommendation system
- 📸 Image-based crop and farm analysis
- 📍 Location-based farming support
- 📅 Seasonal crop suggestions
- 💹 Real-time agricultural market price insights
- 🌐 Multi-language support
- 📱 Fully responsive modern UI
- ⚡ Fast and optimized performance

⚙️ Installation
1. Clone the Repository
git clone https://github.com/YOUR_USERNAME/agro-assist-farm.git
2. Navigate to Project Directory
cd agro-assist-farm
3. Install Dependencies

Using npm:

npm install

Or using bun:

bun install
▶️ Run the Development Server
npm run dev

Application runs on:

http://localhost:5173
🏗️ Build for Production
npm run build

Preview production build:

npm run preview
🌍 Environment Variables

Create a .env file in the project root.

Example:

VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
